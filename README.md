## 📖 项目说明

本项目是在开源项目 **XyhTender/qq-farm-automation-bot** 基础上进行二次开发完成的 QQ 农场自动化辅助工具。

感谢原作者对项目基础能力的开源贡献，也感谢开源社区长期以来提供的思路、反馈与工具生态支持。本项目会继续尊重开源精神，在原有能力基础上补充更适合当前使用场景的功能与体验。

本项目二开代码使用 **Codex** 工具基于 **GPT-5.5** 与 **DeepSeek-v4-flash** 完成。

> ⚠️ 本项目二开部分目前仅在 **Windows** 和 **macOS** 环境测试通过，其他环境不保证可以正常运行。二开功能仍可能存在不完善或异常情况，如使用中遇到问题，还请理解与见谅。

## 🚀 部署、构建与启动

### 前置要求

安装 **Node.js 20+**（[下载地址](https://nodejs.org/)）并启用 pnpm

```bash
node -v
corepack enable
pnpm -v
```

### 安装与构建

在项目根目录执行：

```bash
# 安装依赖
pnpm install

# 构建前端
pnpm build:web
```

### 启动

```bash
# 默认端口启动
pnpm dev:core
```

#### 自定义端口

```powershell
$env:ADMIN_PORT = "你的端口号"
pnpm dev:core
```

### 访问

打开浏览器访问 `http://localhost:3007`

|------|-----|

| 默认密码 | `admin` |
